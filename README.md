# ShopEasyy AI
This is the ShopEasyy AI repository.

## Backend
- Models, controllers, and routes for the backend are located in the `app` directory.

## Frontend
- The frontend source code is in the `src` directory and static assets are in the `public` directory.

## Configuration
- Configuration files are located in the `config` directory.

## Documentation
- Documentation for architecture, deployment, and user guide are in the `docs` directory.
